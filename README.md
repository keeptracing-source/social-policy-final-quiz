# 사회정책 기말 퀴즈 · 암기

기말고사(9~13주차 + 한국 복지국가) 핵심개념을 **빠르게 풀고 반복 암기**하는 모바일 학습 웹앱입니다.
설치 없이 휴대폰 브라우저에서 바로 학습할 수 있습니다.

## 학습 모드
- 📇 **암기카드** — 개념 ↔ 정의 플래시카드. `외웠어요/다시`로 분류 후 *모르는 것만* 반복
- ⭕ **OX 퀴즈** — 함정 위주 30문, 즉시 해설
- 🔢 **객관식** — 4지선다 25문, 정답·해설 표시
- 📝 **서술형** — 10문항 (⭐ = **실제 기말 족보 기출 5선**, 나머지는 예상·연결). 핵심 키워드 + 답안 뼈대(개념정의→비교→국가사례)
- 📊 **진도** — 암기 진도율·정답률 자동 저장(이 기기 브라우저)

## ⭐ 기말 족보 기출 5선 (최우선)
1. (10주) Esping-Andersen 탈상품화 정의 + 세 regime 대표정책 + 탈상품화 수준 *(최다 빈출)*
2. (9주) Pateman 수혜자격 2가지 시스템(two-tier) + 젠더불평등 유지·강화
3. (9주) Wollstonecraft's dilemma + 해결책 + 가장 잘 시행하는 국가 예시
4. (13주) Hook — 남녀 돌봄시간 격차의 사회정책 요소 + 돌봄 평등 잘된 2개국
5. (13주) 가사노동 성역할의 doing gender 논의
